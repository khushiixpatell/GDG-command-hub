# GDG Command Hub

A centralized command and event management platform designed for Google Developer Groups (GDG) organizers to streamline event planning, team coordination, volunteer management, and operational workflows.

Built to help Google Developer Group organizers, like myself, coordinate events, volunteers, and operations from a single AI-powered dashboard.

## Live Demo

https://gdg-command-hub.onrender.com

---

## Overview

GDG Command Hub was built to simplify the administrative and organizational challenges faced by student leaders and community organizers. The platform provides a single dashboard for managing events, tracking activities, coordinating volunteers, and maintaining visibility across multiple initiatives.

The goal is to reduce manual coordination work and provide organizers with a more efficient way to run successful community events.

## Motivation

As student organizations and developer communities grow, managing events and coordinating volunteers becomes increasingly complex. GDG Command Hub was created to provide a lightweight operational dashboard that helps organizers spend less time managing logistics and more time building meaningful community experiences.
